## ðŸŒ Rangkuman & Panduan praktis singkat

* **Paradigma vs Level:** Paradigma (cara berpikir) *berlalu-lintas* di semua level abstraksi. Level (low â†’ very high) menjelaskan seberapa dekat bahasa ke mesin.
* **Jika ingin *memodifikasi engine* â†’ persiapkan:** bahasa implementasi engine, build toolchain, debugging/profiling tools, knowledge domain (runtime/GC/JIT/compiler internals).
* **Jika ingin *menulis plugin* â†’ persiapkan:** API plugin (native embedding API, FFI, LSP, modul system), bahasa host, packaging, testing.
